## Итак, о чем этот проект?

Два Docker-контейнера:
- **Генератор** (Python) → создаёт `data.csv`
- **Аналитик** (Node.js) → создаёт `report.html`

Данные хранятся в папке `data/` на вашем компьютере.

---

## Тема моей работы - Кофейня!

Система содержит 50 заказов:
- Напиток (Латте, Капучино, Эспрессо, Раф)
- Объем (0.2, 0.3, 0.4, 0.6 л)
- Цена (150–500 ₽)
- Сироп (шоколадный, ореховый, солёная карамель, ванильный)

## Теперь перейдем к описанию команд (всё как по заданию)

```bash
# Генератор
./run.sh build_generator      # собрать образ
./run.sh run_generator        # создать data.csv
./run.sh create_local_data    # без Docker (отладка)

# Аналитик
./run.sh build_reporter       # собрать образ
./run.sh run_reporter         # создать report.html

# Вспомогательные
./run.sh structure            # структура проекта
./run.sh clear_data           # очистить data/
./run.sh inside_generator     # что видит генератор
./run.sh inside_reporter      # что видит аналитик
```

## Как запустить и увидеть результат?

Для этого достаточно написать эти команды в таком порядке

```bash
./run.sh build_generator
./run.sh run_generator
./run.sh build_reporter
./run.sh run_reporter
```

И открыть получившийся файл data/report.html в каком-то онлайн HTML-редакторе (возможно у вас получится найти даже сразу по первой ссылке).
